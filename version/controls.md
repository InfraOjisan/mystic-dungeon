# Mystic Dungeon 操作説明書

## 概要

Mystic Dungeon は、ダンジョンを探索し、モンスターと戦いながら
伝説の装備を集めるターン制ダンジョンRPGです。「不思議のダンジョン」シリーズをモチーフとしています。

## 起動方法

```bash
python3 rogue.py
```

## 基本操作

| キー | 動作 |
|------|------|
| `h` / `←` | 左に移動 |
| `j` / `↓` | 下に移動 |
| `k` / `↑` | 上に移動 |
| `l` / `→` | 右に移動 |
| `y` | 左上に移動 |
| `u` | 右上に移動 |
| `b` | 左下に移動 |
| `n` | 右下に移動 |
| `Space` | その場のゴールドを拾う |
| `.` (ピリオド) | 1ターン休息（HPが1回復） |
| `>` | 階段を降りる |
| `i` | インベントリを開く |
| `d` | アイテムを捨てる |
| `m` | 踏破済みマップを表示（ASCII） |
| `s` | ショップを開く（ショップタイル上で） |
| `S` | セーブして続ける |
| `Q` | セーブして終了 |
| `q` | 終了（セーブしない） |

## 戦闘システム

敵がいるマスに移動すると**テキストバトル画面**に遷移します。

### バトル画面操作

| キー | 動作 |
|------|------|
| `e` | 装備中の武器で攻撃 |
| `1`–`9` | インベントリの該当スロットの武器で攻撃 |
| `f` | 素手で攻撃 |

### 戦闘の流れ

1. 武器を選択（接敵側が先攻）
2. ダメージ計算・表示
3. 相手が生存していれば反撃
4. どちらかが倒れるまで繰り返し

### ダメージ計算式

- **物理攻撃**: `max(3, 攻撃側PATK + 武器PATK - 防御側PDEF)`
- **魔法攻撃**: `max(3, 攻撃側MATK + 武器MATK - 防御側MDEF)`
- **弱点属性**: 攻撃力が1.5倍（防御計算の前に乗算）
- **クリティカル**: 防御力無視で `攻撃力 × 会心倍率` のダメージ

### ステータス

| ステータス | 説明 |
|-----------|------|
| PATK | 物理攻撃力 |
| MATK | 魔法攻撃力 |
| PDEF | 物理防御力 |
| MDEF | 魔法防御力 |
| CRIT% | クリティカル発生率 |
| CRIT DMG | クリティカル時の倍率 |

## セーブ／ロード

### セーブスロット
セーブデータは3スロットまで保存できます。
`rogue_save_1.json` / `rogue_save_2.json` / `rogue_save_3.json` に保存されます。

### セーブする
- **`S`**（ Shift + s ）: 現在のスロットに上書き保存して続ける
- **`Q`**（ Shift + q ）: 現在のスロットに上書き保存して終了
- 初回セーブ時は保存先スロット `1/2/3` を選択します
- ステータスバーに `Slot:1` のように現在のスロットが表示されます

### ロードする
ゲーム開始時にセーブデータがある場合、スロット選択画面が表示されます。
`1` / `2` / `3` でロード、`n` で新規ゲーム、`q` で終了。

## タイトル画面

セーブデータがある場合：

| キー | 動作 |
|------|------|
| `1`–`3` | 対応するスロットのセーブデータをロード |
| `n` | 新規ゲーム（難易度選択） |
| `q` | 終了 |

新規ゲーム時：

| キー | 動作 |
|------|------|
| `1` | Easy |
| `2` | Middle（推奨） |
| `3` | Hard |

## インベントリ画面

`i` キーで開きます。

- `1`–`9` でアイテム選択 → 武器/防具は装備交換、ポーションは使用
- `d` で捨てるモード
- `q` / `ESC` で閉じる

レアリティ表示: `+`=Uncommon `★`=Rare `✦`=Legendary

## ショップ

マップ上に `🏪` のショップタイルがあります。その上に立って `s` キーで開きます。

- 武器・防具をゴールドで購入
- 不要な装備を売却（買値の半額）
- ショップの品揃えはフロアごとにランダム

## 装備システム

武器と防具は同時に装備できます。

- 武器: PATK/MATK/会心率/会心倍率 を強化
- 防具: PDEF/MDEF を強化
- 装備品にはレアリティ（Common/Uncommon/Rare/Legendary）があり、高いほど強力

## レアリティ一覧

| レア度 | タグ | 特徴 |
|--------|------|------|
| Common | （なし） | 基本性能のみ |
| Uncommon | `+` | 若干の追加ボーナス |
| Rare | `★` | 大きなボーナス |
| Legendary | `✦` | 非常に強力。固定名称 |

## アイテム入手方法

- **ゴールド**: マップ上に落ちている（自動取得）
- **武器・防具**: 敵を倒した時にドロップ、またはショップで購入
- **回復薬・食料**: 敵を倒した時にドロップ
- **レアアイテム**: 各フロアに1体いる「レアベアラー」の敵を倒すと確定ドロップ

## 強くてNewGame

ゲームオーバー時に選択できます。

| コマンド | 内容 |
|---------|------|
| `r` | 最初からやり直す（完全リセット） |
| `s` | **強くてNewGame**: 現在のHP・ステータス・所持品を維持して再開 |
| `q` | 終了 |

何度も挑戦して伝説装備をコレクションしましょう。

## マップ表示

`m` キーで踏破済みエリアをASCII文字で一覧表示します。
```
@=You  #=Wall  .=Floor  >=Stairs  A-Z=Monsters
```

## 戦闘のヒント

- 敵の弱点属性を突くと1.5倍ダメージ
- 魔法攻撃は物理防御の高い敵に有効（逆も同様）
- Legendary武器の収集がゲームの主目標
- レアベアラー（赤いオーラの敵）を確実に倒そう
- `S` キーでこまめにセーブ
