# Mystic Dungeon 操作説明書 (ブラウザ版)

## 概要

Mystic Dungeon は、ダンジョンを探索し、モンスターと戦いながら
伝説の装備を集めるターン制ダンジョンRPGです。「風来のシレン」「トルネコ」シリーズをモチーフとしています。

## 起動方法

```bash
python3 -m http.server 8080 --directory /path/to/game
```
ブラウザで `http://localhost:8080` を開く。サーバ不要で `index.html` を直接開いても動作します。

## 画面構成

- **マップ画面**: タイル＋絵文字で表示。見える範囲のみ照らされ、一度踏破した場所は暗く表示
- **ステータスバー**: 画面下部にHPバー、攻撃力、ゴールド、階層、装備を表示
- **メッセージ**: ステータスバー下部に直近のログを表示

## 戦闘システム

敵と同じマスに移動すると**バトル画面**に遷移します。ターン制でプレイヤーと敵が交互に攻撃します。

### バトル画面操作

| キー | 動作 |
|------|------|
| `e` | 装備中の武器で攻撃 |
| `1`–`9` | インベントリの該当武器で攻撃 |
| `f` | 素手で攻撃 |

### 戦闘の流れ

1. 接敵した側が先攻で武器選択
2. ダメージ計算と表示
3. 相手が生存していれば反撃
4. どちらかが倒れるまで繰り返し

### ダメージ計算

- 物理攻撃: `max(3, PATK + 武器PATK - 敵PDEF)`
- 魔法攻撃: `max(3, MATK + 武器MATK - 敵MDEF)`
- 弱点属性: 防御計算前に攻撃力1.5倍
- クリティカル: 防御無視で `攻撃力 × 会心倍率`
- 最低保証ダメージ: **3**（どんなに防御が高くても3は通る）

### ステータス

| 項目 | 説明 |
|------|------|
| PATK / MATK | 物理/魔法攻撃力 |
| PDEF / MDEF | 物理/魔法防御力 |
| CRIT% | クリティカル発生率 |
| CRIT DMG | クリティカル倍率 |

## セーブ／ロード

セーブデータはブラウザの `localStorage` に保存されます（キー名: `md_save_1`～`3`）。ファイル保存は不要です。

- **`S`**: 現在のスロットに上書き保存
- **`Q`**: 保存して終了
- 初回セーブ時はスロット `1/2/3` を選択

タイトル画面でセーブデータがある場合、`1/2/3` でロード、`n` で新規ゲーム、`q` で終了できます。

## タイトル画面

| キー | 動作 |
|------|------|
| `1`–`3` | 該当スロットをロード / 新規は難易度選択 |
| `n` | 新規ゲーム（セーブあり時） |
| `q` | 終了 |

新規ゲーム時の難易度:
| 難易度 | 特徴 |
|--------|------|
| `1` Easy | 10Fにボス出現。倒せばクリア。金貨3倍、ショップ半額 |
| `2` Middle | 100Fにボス出現。標準バランス |
| `3` Hard | エンドレス。最高到達階数と挑戦回数を記録 |

## インベントリ

`i` で開く。スロットは数字キーで選択:
- 武器: 装備中のものと交換
- 防具: 装備中のものと交換
- ポーション: 使用（HP回復）
- `d`: 捨てるモードに切り替え
- `q` / `ESC`: 閉じる

レアリティ表示: `+`=Uncommon `★`=Rare `✦`=Legendary

## ショップ

フロア生成時に35%の確率で出現。`🏪`の上に立って `s` キーで開く。

- 武器・防具をゴールドで購入
- 不要な装備を買値の半額で売却
- 品揃えはフロアごとにランダム（レアリティは深度に応じて変動）

## 装備

武器と防具を同時に装備できます。装備品にはレアリティが設定され、高いほど強力なボーナスが付きます。

| レア度 | タグ | ボーナス |
|--------|------|----------|
| Common | なし | 基本性能のみ |
| Uncommon | `+` | 若干の追加ボーナス |
| Rare | `★` | 大きなボーナス |
| Legendary | `✦` | 最大ボーナス＋固定名称 |

### レジェンダリー武器

Muramasa / Excalibur / Zantetsuken / Shuriken / Gungnir / Mjolnir

### レジェンダリー防具

Dragon Mail / Aegis

## アイテム入手

- **ゴールド**: マップ上に落下（歩けば自動取得）
- **武器・防具**: 敵ドロップ or ショップ購入
- **レア枠**: 各フロアに1体いる「レアベアラー」（HPが周囲より高い）を倒すと高確率でRare以上確定ドロップ

## 強くてNewGame

ゲームオーバー時に選択:

| キー | 内容 |
|------|------|
| `r` | 完全リセット（最初から） |
| `s` | **強くてNewGame**: 現在のHP・攻撃力・所持品を維持して再開。ゴールドやレア装備を引き継いで再挑戦できる |
| `q` | 終了 |

## マップ表示

`m` キーで踏破済みエリアを一覧表示。

```
@=プレイヤー  #=壁  .=床  >=階段  A-Z=モンスター
```

## ヒント / 戦略

- 序盤は素手でも戦える弱い敵（Ant/Bat）を探して倒し、装備を拾おう
- 敵の弱点属性を突くと1.5倍ダメージ。魔法が通らない敵には物理、物理が通らない敵には魔法
- 最低3ダメージは必ず通るので、高防御の敵にも削りは入る
- `.`（ピリオド）でHPを1回復できる。安全な場所で小まめに回復
- レアベアラーは他のモブよりHPが多い。見分けて確実に倒し、レア装備を狙おう
- 強くてNewGameを活用すればゴールドやレア装備を蓄積してジワジワ強くなる
